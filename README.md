# 🍽️ Big Plate Restaurant Chatbot

A simple **rule-based AI chatbot** built with Python that helps customers interact with a restaurant digitally. The chatbot provides restaurant information, menu details, food recommendations, and online ordering support.

---

## ✨ Features

- 🏠 View restaurant information
- 📞 Get contact details
- 🌐 Access website and social media links
- 📍 View restaurant branches
- 🍔 Browse menu by category
- 🔎 Get food item details:
  - Price
  - Description
  - Ingredients
  - Preparation time
  - Vegetarian information
- 🛒 Open online ordering pages

---

## 🧠 How It Works

The chatbot uses a **rule-based NLP approach** to identify user queries and generate responses using a predefined knowledge base.
